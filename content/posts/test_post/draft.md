## Giới thiệu

Python là một ngôn ngữ lập trình mạnh mẽ và dễ học. Trong bài viết này, chúng ta sẽ
cùng nhau khám phá những khái niệm cơ bản nhất để bắt đầu hành trình lập trình.

## Tại sao nên học Python?

Có rất nhiều lý do để chọn Python làm ngôn ngữ đầu tiên của bạn. Python được sử dụng
rộng rãi trong trí tuệ nhân tạo, khoa học dữ liệu và phát triển web.

Bạn có thể tìm hiểu thêm tại [trang chủ chính thức của Python](https://www.python.org)
hoặc xem ![hình minh họa](https://example.com/python-logo.png) để biết thêm chi tiết.

## Cài đặt môi trường

Trước tiên, hãy tải Python từ [trang tải xuống](https://www.python.org/downloads/) về máy.

## Ví dụ đầu tiên

Đây là chương trình Python đơn giản nhất:
```python
# In ra màn hình
print("Xin chào thế giới!")

for i in range(5):
    print(f"Dòng số {i}")
```

## Các kiểu dữ liệu cơ bản

Python có nhiều kiểu dữ liệu khác nhau như số nguyên, số thực, chuỗi ký tự và danh sách.
Việc hiểu rõ các kiểu dữ liệu này sẽ giúp bạn viết code hiệu quả hơn rất nhiều.
